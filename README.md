1. Install Prerequisites

.NET SDK — download from dotnet.microsoft.com
VS Code — download from code.visualstudio.com


2. Install the C# Extension in VS Code

Open VS Code
Click the Extensions icon (left sidebar, looks like 4 squares)
Search "C#" — install the official one by JosKreativ


3. Download Your Code from GitHub (as ZIP)

Go to https://github.com/GMCC-CPE/DormMarket
Click the "Code" button beside add file
Click "Download ZIP"
Extract the ZIP to a prefer file path


4. Open the Project in VS Code

File → Open Folder → select the extracted DormMarket folder


5. Open new terminal button beside run at the top


6. Run the Project

type in terminal: dotnet run -> click https://localhost link or type in terminal: dotnet watch

7. Shutdown the system
Press ctrl + C in terminal
